# location

### 简介
这是一个基于 matplotlib 开发的地图可视化工具。它能将 ROS2 产生的 .pgm 格式地图及其对应的 .yaml 配置文件直观地显示出来。

### 功能
*   加载并渲染地图图像。
*   自动标出坐标轴与原点位置。
*   支持通过鼠标点击在地图上标记特定坐标点。
*   按下键盘 'c' 键可一键清除所有已标记的点。

### 参数
*   map_yaml_path: 指定地图 yaml 文件的绝对路径。

### 使用方法
运行以下指令启动可视化脚本：
```bash
ros2 launch location map_visualizer_launch.py
```
该工具常用于在地图上快速拾取导航点坐标。获取的坐标可直接填入行为树的目标位置配置中。
